iRail terminology

Terminology used throughout the iRail project can be found here:

Occupancy

  • terms/unknown. Unknown: there is insufficient information to define an occupancy level.
  • terms/low. Low occupancy: there are plenty of seats left.
  • terms/medium. Medium occupancy: it is hard to find a seat and it is difficult to sit together.
  • terms/high. High occupancy: there are no seats left and people have to stand up.
